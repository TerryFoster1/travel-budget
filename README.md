# Travel Budget

Production-ready scaffold for the Travel Budget platform.

Travel Budget is a travel membership and travel opportunity platform. This repository currently contains architecture placeholders only. It intentionally does not implement payments, banking, booking systems, travel APIs, or authentication providers.

## Workspaces

- `client`: React, TypeScript, Vite, Tailwind frontend shell.
- `server`: Node.js, Express, TypeScript API shell.
- `prisma`: PostgreSQL and Prisma planning surface.
- `docs`: Product and architecture planning documents.
- `scripts`: Operational and developer automation placeholders.
- `tests`: Vitest and Playwright test organization placeholders.
